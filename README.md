This is a project using MERN stack (mongodb,rectjs,node.js,react.js) in which food delivery site is made.In this project there are many functionalities like login and logout page,explore menu, add to cart and remove from cart in which user can add or remove food items from cart.In this project , I use jwt tokens for authentication.In this there are other functionalities like stripe payment gateway,order page,admin panel,admin can add food products,he/she can view order page and even change the order delivery status.This projects explores various new ideas.
